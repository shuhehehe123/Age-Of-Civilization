# Age-Of-Civilization（文明时代）

## 简介

这是一个我的世界Minecraft原版数据包，含有人类从诞生以来到维多利亚时代（1832年）以前的多个方面。

成就分为多个方面：如探索，宗教与文化等。

## 快速游玩

此数据包安装过程和其他数据包一致，但需要将数据包文件：.zip放入世界材质包（资源包）文件夹。
